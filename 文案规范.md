# 文案规范（v1）

> 说话的人是"这本日记"，不是助手、不是教练、不是客服。它安静、诚实、不评价、不催促。

---

## 一、语气原则

1. **陈述句为主**。全产品禁用感叹号，禁用 emoji。唯一允许的问句是回访提示（"感觉如何？"）。
2. **称呼"你"**，产品不自称"我"或"我们"，不说"请"。
3. **叙事时间，不是数据时间**："三个月""上个秋天""犹豫了一个月"，而不是"90 天""2026-04-20"。精确日期只出现在详情页的日记条目和元数据里。
4. **不评价、不夸奖、不催促**。禁止"太棒了""加油""别忘了""快来"。产品对用户的决定没有立场——买与不买同样被平静对待。
5. **动词朴素**：记下、想清楚、放下、陪伴、回访。禁止"解锁""开启""管理""打卡""达成"。
6. **数字只在叙事里合法**："你认真想过 12 件东西"可以，"总资产 ¥6,973"不可以。数的永远是思考和时间，不是钱和物。

## 二、词汇表（必须统一）

| 用这个 | 不用这个 |
|---|---|
| 犹豫中 | 待购买、心愿、想要 |
| 陪伴你 | 已使用、服役中、持有中 |
| 想清楚了，没有买 | 已放弃、已取消 |
| 搁置了 | 逾期、未完成、闲置 |
| 回访 | 打卡、任务、待办 |
| 索引 | 库存、资产、物品管理 |
| 记下一个念头 | 添加物品、新建资产 |
| 追记 | 更新、编辑 |
| 起点 | 创建、录入 |

## 三、各场景文案

### 状态叙事（卡片第二行）
- 犹豫中 · 第 23 天
- 陪伴你 · 三个月 / 一年两个月
- 想清楚了，没有买 · 六月
- 搁置了
- 刚开始陪伴你（购买未满一个月，替代"上次回访 · 无"）

### 首页头部（陈述句状态）
- 你正在考虑两件东西
- 最近很安静（无犹豫中物品时）

### 回访提示（首页日记体一行 + 推送共用）
- 降噪耳机已经陪你三个月了，感觉如何？
- 落地灯陪伴你一年了。想写点什么吗
- 那台相机你想了一个月了。现在怎么想？

### 仪式时刻（全屏，一句话为主体）
- 第一次记录：这是你的第一条决策记录。慢慢来。
- 决定购买：犹豫了 23 天，你决定带它回家。
- 决定不买：这个念头陪了你 47 天，你想清楚了。
- 周年：降噪耳机陪伴你一年了。

### 空状态
- 时间线：这一页还是空白。想到什么，就从它开始。
- 索引（筛选无结果）：这一页还是空白。
- 详情页无回访：还没有回访记录。日子还长。

### 索引头部（随筛选变化）
- 全部：你认真想过 12 件东西
- 陪伴中：8 件东西正陪伴着你
- 犹豫中：你正在考虑 2 件东西
- 没有买：你想清楚后放下了 2 件东西

### 按钮（动词开头，1–4 字）
- 记下 / 先放着 / 想清楚了 / 带它回家 / 写回访 / 返回

### 系统文案（直说，不道歉不卖萌）
- 删除确认：删除后，这段记录不可恢复。 → 删除 / 先留着
- 同步失败：暂时没能同步到 iCloud，记录已保存在本机。
- 通知权限：回访提醒需要通知权限。每周至多一条。

## 四、禁用清单

感叹号 · emoji · 成功/恭喜 · 立即/马上/快 · 仅需/免费/限时 · 解锁/开启/达成 · 打卡/连续 X 天 · 省下了 X 元 · 任何自称（"我们为你准备了……"）
